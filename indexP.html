Top ventas 95% valoraciones positivas(775)

Aprende de forma detallada cómo funciona Canva mientras practicas diseñando tu propia guía de estilo y una página web completa

Porfolios, presentaciones, carteles, videos, publicaciones para redes sociales o incluso la maquetación de una página web. En este curso online, la fundadora de la agencia de diseño gráfico Aplum Studio, Claudia Cánovas, te enseñará cómo hacer todo esto utilizando una misma plataforma: Canva.

Así pues, a lo largo de las unidades, aprenderás el funcionamiento de Canva de 0 a 100 para crear contenido visual de forma profesional. Verás la importancia del diseño para contar una historia y acercarte a tu público ideal y comprobarás que eres capaz de potenciar una identidad estética de forma esencial, pero con gran calidad.
